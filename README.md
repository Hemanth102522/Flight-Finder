# ✈️ Flight Booking System (MERN Stack)

A full-stack **Flight Booking Web Application** built using the **MERN Stack (MongoDB, Express, React, Node.js)**.

This application allows users to search available flights, book tickets, view bookings, and cancel bookings.
Admins can add new flights to the system.

---

# 🚀 Features

* Search available flights
* Book flight tickets
* View booked flights
* Cancel bookings
* Admin panel to add flights
* MongoDB Atlas database integration
* Full MERN stack architecture

---

# 🛠 Tech Stack

### Frontend

* React.js
* Bootstrap
* Axios

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas

# ⚙️ Installation & Setup

Clone the repository

```
git clone https://github.com/vasanth1931v/flight-booking-app.git
```

---

# Backend Setup

Navigate to server folder

```
cd server
```

Install dependencies

```
npm install
```

Start backend server

```
node index.js
```

Backend runs on:

```
http://localhost:6001
```

---

# Frontend Setup

Open another terminal and navigate to client folder

```
cd client
```

Install dependencies

```
npm install
```

Run the React application

```
npm start
```

Frontend runs on:

```
http://localhost:3000
```
